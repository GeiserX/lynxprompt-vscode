# Changelog

All notable changes to the LynxPrompt VS Code extension will be documented in this file.

## [0.1.0] - 2026-03-03

### Added

- Sidebar tree view with "My Blueprints" and "Local Config Files" sections
- Device flow authentication (Sign In / Sign Out)
- Pull blueprints from cloud to workspace
- Push local AI config files to LynxPrompt
- Diff view comparing local files with cloud blueprints
- Auto-detection of AI config files (CLAUDE.md, AGENTS.md, .cursor/rules/, copilot-instructions.md, .windsurfrules)
- File watchers for change notifications on linked config files
- Status bar connection indicator
- Generate Config command (opens wizard in browser)
- Convert Format command for local format conversion
- Configurable API URL for self-hosted instances
- SecretStorage-based token persistence
